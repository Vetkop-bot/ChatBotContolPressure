# Бот контроля артериального давления

Telegram-бот для ввода показаний давления (систола, диастола, пульс).  
На данный момент работает в режиме распознавания чисел из текста и фото, сохранение истории в PostgreSQL. 
В дальнейшем будет добавлен полный анализ: расчёт индексов, зоны риска, сохранение истории и напоминания о приёме препаратов.

## Возможности (текущие)

- Приём сообщений с тремя числами (разделители: пробел, запятая, точка с запятой, слеш, двоеточие)
- Распознавание систолы, диастолы, пульса
- Ответ с распознанными значениями
- Распознавание по фото 
- Сохранение в БД 	
- Логирование сообщений в консоль сервера
- Команды 
`/start` 
`/help` 
`/stats` 

## Технологии

- **Node.js**
- **TypeScript**
- **Telegraf** – фреймворк для Telegram Bot API
- **Express** – для приёма вебхуков MAX
- **Prisma** – ORM для работы с PostgreSQL
- **PostgreSQL** – база данных (запускается в Docker)
- **Tesseract.js** – OCR для распознавания чисел на фото
- **Sharp** – обработка изображений (обрезка, изменение размера, контраст)
- **Docker** – для запуска PostgreSQL

## Установка и запуск

### Клонируй репозиторий
git clone https://github.com/Vetkop-bot/ChatBotContolPressure.git
cd ChatBotContolPressure
### Установи зависимости
npm install
### Создай файл .env в корне проекта
BOT_TOKEN= 
DATABASE_URL=
MAX_BOT_TOKEN=
MAX_WEBHOOK_SECRET
### Запуск PostgreSQL через Docker
docker compose up -d
### Применение мишрацций Prisma
npx prisma generate
npx prisma migrate dev --name init
### Запусти ботов
npm run dev
